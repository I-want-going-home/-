1. 사용 gcc버전 : 14.1.0

2. 사용 개발 툴 : VSCode

3. 출력방식 : 일반 문자열 : printf(), 문자열 받기 : scanf()

4. (보너스) 날짜와 이름 입력 후 3초뒤 화면을 지우고 스플래시 화면 표시

=> stdio.h, stdlib.h 헤더파일들을 제외한 헤더를 추가하지않는 방법으로 _sleep(3000)사용하여 3초간 대기 설정
=> c언어에서 사용가능한 콘솔화면초기화 명령어인 system("cls")를 사용하여 화면초기화

5. while (getchar() != '\n'); , getchar() 사용이유 
=>  .exe 파일에 콘솔화면이 결과를 보여주기 전에 종료, 이를 방지하기 위해 키입력 후 종료하게 끔 설정
=> while (getchar() != '\n')는 공백이 들어올경우 바로 넘어가기 때문에 공백은 받지않게끔 설정

6. 시간 정리방법

=> date[11]에 저장되어있는 문자열 형태의 yyyy-mm-dd를 정수형으로 변환하기 위해 atoi() 함수 사용
=> atoi() 함수는 숫자로 읽을 수 없는 문자열에서 멈추므로 -를 기준으로 끊김
=> 이를 활용하여 년도는 year = atoi(date) << yyyy
=> 달은 month = atoi(date + 5) << mm (5는 달이 시작되는 배열 위치)
=> 일은 day = atoi(date + 8) << dd (8은 일이 시작되는 배열 위치)

7. 사용자, 실행 시간 출력

=> 사용자는 문자열을 그대로 받아쓰기때문에 %s를 사용
=> 실행 시간은 문자열을 정수형으로 변환했기때문에 %d를 사용하여 각각 나눠 %d년 %d월 %d일로 지정해 각각 년도, 월, 일 을 대입함

8. 삼각형의 for문 사용

=> *표에 대한 직각삼각형, 역직각삼각형 방법은 triangle.txt 파일과 reversetriangle 파일에 각각 저장해놓음
=> 실제 적용된 함수에는 String.h 헤더파일이나 다른 헤더파일을 추가할 수 없으므로 문자열 복사가 불가능하다고 판단,
     배열에 *을 담아 문자열을 합치는 방향으로 진행함, 반복문을 사용하여 배열의 순번을 지정하여 직각삼각형, 역직각삼각형으로 적용되게 끔 만듦.
